# Pranto Deb Premium Portfolio

Vue 3 + Vite single page portfolio with premium white UI, real project showcase, mobile-first hero ordering, app publishing showcase, CV download, social links, and deployment files.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Upload the `dist` folder to Netlify or cPanel.

## Included

- Laravel/Vue/PHP portfolio sections
- Real work previews
- Shotayu Android app / Google Play showcase
- Experience timeline
- App development & publishing skill/service
- WhatsApp, GitHub, LinkedIn, Facebook links
- CV download
